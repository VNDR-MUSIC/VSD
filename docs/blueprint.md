# **App Name**: VSD Network

## Core Features:

- Dashboard: Centralized dashboard for VSD token information and resources.
- Token Information: Display key information about VSD, the decentralized stablecoin, in a clear and accessible format.
- Ecosystem Projects: Showcase the different projects currently using the VSD token in its ecosystem.
- Developer Resources: Dedicated section for developer-related resources, documentation, and community links.
- Legal Footing: A Footer with links to legal documentation such as terms and conditions, privacy policy and similar
- AI-Powered Documentation Search: AI-powered search that provides responses and directs users to relevant sections in documentation using an LLM tool.

## Style Guidelines:

- Primary color: Deep red (#D00039) for VSD brand recognition, aligning with the specified --vsd-red CSS variable.
- Background color: Very dark gray (#000000), as defined by the --background CSS variable for a modern, minimalist look.
- Accent color: Medium gray (#808080) as defined in the --accent CSS variable for a clean UI with balanced contrast.
- Body and headline font: 'Orbitron' (sans-serif) for a futuristic and tech-inspired feel, per the original layout.tsx.
- Use simple, geometric icons that reflect the decentralized and technical nature of the VSD hub. Ensure icons are consistent with the site's dark theme.
- Modern, minimalist layout with clear sections for easy navigation. Use of whitespace to improve readability.
- Subtle transitions and hover effects on interactive elements. For instance the buttons should follow the effect in globals.css.
